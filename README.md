# LeetCode Exercises

This repository contains my solutions to various [LeetCode](https://leetcode.com) coding challenges, along with JUnit tests to verify their correctness.

## Solutions

The solutions can be found in the [Solutions](./Solutions) folder. Each solution is in its own subfolder, named after the LeetCode problem it solves.

## Tests

The JUnit tests for each solution can be found in the [Tests](./Tests) folder. These tests verify that each solution works correctly for a variety of inputs.

If you find any bugs or have any suggestions for improvements, feel free to open an issue or a pull request. Your input is always appreciated!
